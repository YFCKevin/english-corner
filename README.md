<h1 align="center">TalkYo</h1>
<p align="center">
模仿 TalkMe App 製作的英文口說智慧學習 WebApp，讓你能與 AI 導師一對一對話，結合語音分析、文法校正與實用情境訓練，幫助你自信開口說英文。
</p>

---

## 🔧 功能

1. 提供 14 位風格各異的 AI 英語導師。
2. 可進行語音對話、逐句發音練習與語音回饋。
3. 支援圖片語義分析與文字文法修正。
4. 整合多種真實情境練習（如與朋友聊天、面試、客訴處理等）。
5. 可儲存收藏語句，並回顧練習紀錄。
6. 支援產生、命名、刪除及分享對話標題。
7. 可視化追蹤學習進度，打造個人化學習路徑。
8. 所有功能皆支援手機端操作。

---

## 🧰 技術

- **後端**：Spring Boot
- **前端**：Alpine.js、Bootstrap、Jquery
- **語言**：Java 17、JavaScript
- **資料庫**：MongoDB
- **即時通訊**：WebSocket
- **雲端部署**：Google Cloud Run + Storage
- **第三方服務**：
  - OpenAI GPT-4o-mini
  - Gemini 1.5 Flash
  - Microsoft Azure Speech to Text / Text to Speech
