<h1 align=center>TalkYo</h1>
<p align=center>模仿 TalkMe App 製作的英文口說智慧學習 WebApp，能與 AI 導師進行一對一對話，並提供發音分析、語音回聽、進階文法和語句查詢和圖片分析，還有搭配多種實用情境，如和朋友訴苦、第一次約會、處理客戶投訴等等，讓口說英文更有自信！

## 技術：
  - **後端：** Spring Boot
  - **前端：** Alpine.js
  - **語言：** Java 17, JavaScript
  - **資料庫：** MongoDB
  - **即時通訊：** WebSocket
  - **雲端部署：** Google Cloud
  - **第三方 API：** openAI (gpt-4o-mini), Gemini-1.5-Flash, Microsoft Azure Audio Service
